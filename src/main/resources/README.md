# Coupon API

API de cupons utilizando Spring Boot.

---

# Tecnologias

- Java 21
- Spring Boot
- H2 Database
- Swagger
- JUnit

---

# Como executar

## Clonar projeto

```bash
git clone URL_DO_REPOSITORIO
```

---

## Rodar aplicação

```bash
mvn spring-boot:run
```

---

# Swagger

http://localhost:8080/swagger-ui/index.html

---

# H2 Console

http://localhost:8080/h2-console

## JDBC URL

```text
jdbc:h2:mem:cupons
```

## Usuário

```text
sa
```

## Senha

```text
password
```

---

# Funcionalidades

- Criar cupom
- Buscar cupom
- Deletar cupom

---

# Testes

Para executar:

```bash
mvn test
```
